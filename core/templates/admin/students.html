ℹ️ How Manual Semester Transitions Work

Select the semester name and year you want to activate, then press Transition to Next Semester. The Reset Agent will: (1) compile and archive final stats for the current semester, (2) generate an AI narrative summary, (3) clear all student responses so every student retakes the chatbot, (4) delete all appointments and events, and (5) set your chosen semester as the new active semester. This action cannot be undone.
